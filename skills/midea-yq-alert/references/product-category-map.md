# 产品类别与业务线映射

本文件来自客户核对后的最新版《副本美的舆情打标规则.docx》产品类别表，用于核准产品对应的四级业务线、三级事业部和二级标签。处理C端、子品牌、中国区域营销与服务、B端业务时，产品归属优先参考本文件。

## 使用原则

- 先用产品关键词匹配四级业务线，再结合风险类型决定二级标签和五级标签。
- 同一产品在普通C端和子品牌行均出现时，若文本明确提到 COLMO、小天鹅、华凌、东芝、WAHIN、Coolfree、酷风、比佛利等子品牌，优先使用 `子品牌风险舆情`。
- 服务售后问题即使匹配具体产品线，二级标签仍使用 `中国区域营销与服务舆情`。
- 价保、补贴、活动承诺、优惠券、差价等价保/营销类争议，按最新版客户表归属 `中国区域营销与服务舆情`，五级标签统一使用 `内销业务`。
- 下表没有的C端产品，或无法分清业务线、同时涉及多个业务线产品时，四级业务线使用 `中国区域`；美美家园等C端服务链路也使用 `中国区域`。
