# 输出模板

本文件定义客户报送、不报送、重复和人工复核的输出格式。

## 需要报送

```text
结论：报送
重复状态：首次/重复/同投诉新进度/同话题传播链
推送标题：【等级】【一级标签】【事业部】【业务线】【舆情分类】
发布时间：{发布时间}
发布平台：{发布平台}
发布人：{发布人}
发布ID：{发布ID输出值}
IP属地：{IP属地；无则写"无"}
链接：{链接}
摘要：{精简后的摘要，突出美的主体、核心问题、诉求/争议点}
传播情况：阅读：{阅读}；点赞：{点赞}；转发：{转发}；评论：{评论}；收藏：{收藏}；粉丝数：{粉丝数}；转载量：{相似文章数}
【AI总结】{一句话总结事实}
【AI研判】{站在美的客户视角说明风险、传播态势、是否高敏或潜在影响}
```

若无传播数据，保留字段并写 `无` 或 `0`。

小红书报送正文外另加提醒：

```text
小红书发布ID需手动核验修改，链接需替换为xhslink.com短链接。
```

## 不报送

```text
结论：不报送
原因：主体不相关/中性信息/广告引流/重复采集/海外中性/噪声/仅背景提及/普通行情
去重键：{若有}
说明：{一句话说明为什么负面或敏感没有落到美的}
```

说明必须可解释，推荐句式：

```text
不报送，因为原文主要讨论{真实主体}，美的仅为{背景/列表/地址/正向对比}，负面未落到美的。
```

## 暂不重复报送

```text
结论：暂不重复报送
重复类型：同链接/同投诉编号/同内容签名/同传播链
已报送记录：{日期/标题/链接/编号}
本次变化：无新增事实/新增状态/新增传播/新增诉求
处理建议：合并备注/作为进展更新报送/忽略重复采集
```

若有新增事实、金额扩大、诉求升级或媒体介入，可改为报送进展更新。

## 需人工复核

```text
结论：需人工复核
原因：主体不明/正文缺失/链接无法核验/系统字段矛盾/标签池不匹配/疑似重复但台账不足
关键信息：{标题、摘要或正文中的关键疑点}
建议核验：{需要人工确认的字段或事实}
```

人工复核不是失败，而是保护交付质量。遇到证据不足、主体不明、标签无法落池、内容互相矛盾时优先复核。
