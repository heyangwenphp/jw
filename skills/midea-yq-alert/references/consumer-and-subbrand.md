# C端产品、中国区域服务与子品牌打标

本文件用于处理消费者产品、售后服务、中国区域、营销权益、子品牌和C端竞对类舆情。

产品对应业务线优先参考 [product-category-map.md](product-category-map.md)。本文件只保留高频边界和打标规则；当产品归属不确定时，先查客户最新版产品类别表。

## C端主体优先级

先看负面落点，不要只看文本里是否出现“品牌”“口碑”“声誉”。

优先级：

1. 明确是子品牌（COLMO、小天鹅、华凌、东芝白电/厨热/微烤、WAHIN、Coolfree、酷风、比佛利等美的体系子品牌）的，优先归 `子品牌风险舆情`。
2. 服务售后统一归 `中国区域营销与服务舆情`，不因出现具体产品线就改成 `C端产品风险舆情`。
3. 产品质量导致的售后、维修、换机、保修、退换流程，归 `产品售后`。
4. 单纯产品自身异常或缺陷，且未进入售后处理链路，归 `产品质量`。
5. 明确属于C端产品或服务链路，但无法识别具体业务线、产品不在客户表内，或同时涉及多个业务线产品：四级业务线归 `中国区域`。
6. 只有年度经营、财报、集团战略或集团品牌整体声誉，且不落到产品体验：转集团规则。

## 客户校准硬规则

以下规则优先于一般产品线判断：

1. **产品质量导致的售后 = `产品售后`**  
   只要内容核心不是单纯描述“产品坏了”，而是已经进入维修、换机、保修、退货退款、售后上门、售后收费、多次维修未果等处理链路，即使起因是产品质量，也将末级分类定为 `产品售后`。

2. **明确子品牌 = `子品牌风险舆情`**  
   明确涉及 COLMO、小天鹅、华凌、东芝白电/厨热/微烤、WAHIN、Coolfree、酷风、比佛利等美的体系子品牌时，一级标签优先使用 `子品牌风险舆情`。不要因为能识别洗衣机、空调、厨热等具体产品线，就改挂 `C端产品风险舆情`。

3. **服务售后 = `中国区域营销与服务舆情`**  
   客服、安装、门店、官方渠道、售后人员、服务态度、推诿、拖延、拒绝处理、乱收费、履约处理等服务售后问题，一级标签统一使用 `中国区域营销与服务舆情`。业务线可保留具体产品线或 `中国区域`，但一级标签不得写成 `C端产品风险舆情`。

## 具体产品线标签

家用空调、冰箱、洗衣机、小家电、厨热、微烤等具体产品问题，使用：

```text
【{等级}】【C端产品风险舆情】【智能家居事业群】【{具体业务线}】【{舆情分类}】
```

示例：

```text
【异常事件】【C端产品风险舆情】【智能家居事业群】【家用空调】【产品质量】
【危机】【C端产品风险舆情】【智能家居事业群】【冰箱】【产品售后】
【异常事件】【中国区域营销与服务舆情】【智能家居事业群】【洗衣机】【服务售后】
```

厨热/微烤边界：

- 烤箱、微波炉、蒸烤箱、微蒸烤一体机、水波炉固定优先归 `微烤`。
- 热水器、燃气热水器、烟机、灶具、洗碗机、集成灶、净水机、管线机归 `厨热`。

## 中国区域标签

`中国区域` 不是“国内舆情”的泛称，而是客户标签体系中的四级业务线。

使用场景：

- 明确属于C端产品或服务链路，但无法确认具体产品线。
- 下表没有的C端产品，或多个产品被同一问题一起投诉，无法归入单一业务线。
- 未提及具体产品，只指向美的整体客服、售后、门店、安装、渠道或服务体系。
- 12315等平台只显示“其他销售服务/其他售后服务问题”，主体为美的相关公司或渠道。
- 美美家园服务链路、交付链路、售后链路出现负面。

标签：

```text
【{等级}】【中国区域营销与服务舆情】【智能家居事业群】【中国区域】【{舆情分类}】
```

## 产品质量、产品售后、服务售后三分法

### 产品质量

产品自身异常或缺陷：

- 漏水、异响、不开机、不制冷。
- 爆裂、漏电、冒烟、爆炸、自燃。
- 材料缺陷、耐用性差、使用后故障。

标签末级：`产品质量`。

若内容已经进入售后处理链路，如维修、换机、保修、退货退款、售后上门、售后收费、多次维修未果，即使起因是产品故障，也改归 `产品售后`。

### 产品售后

产品质量导致的售后问题，或产品发生问题后围绕维修、换机、保修、退换流程的争议：

- 多次维修无果。
- 保修期内收费。
- 申请换机、退货、退款流程推进中。
- 维修后仍未解决。
- 产品故障后进入售后处理链路。
- 产品质量问题导致退换货、保修、售后收费、售后上门或多次维修。

标签末级：`产品售后`。

### 服务售后

服务态度、处理方式、安装、客服、门店、拒绝履约等问题：

- 客服不回复、推诿、拖延、失联。
- 安装师傅乱收费、安装不规范。
- 明确拒绝退货退款、拒绝处理、拒绝价保。
- 官方渠道、门店、京东/天猫旗舰店处理不当。

标签末级：`服务售后`。

一级标签固定使用 `中国区域营销与服务舆情`。例如：

```text
【{等级}】【中国区域营销与服务舆情】【智能家居事业群】【{具体业务线或中国区域}】【服务售后】
```

旧标签 `售后服务` 必须改为 `服务售后`。

## 价格权益与营销争议

价格权益、活动和营销争议按客户最新版 Word 标签表处理。

- 价保、补贴、活动承诺、优惠券、差价。
- 宣传承诺未兑现、活动规则争议。
- 会员活动、美粉活动、营销组织问题。

客户最新版表格的风险点说明为：价保/营销类归属到 `中国区域营销与服务舆情`，统一使用 `内销业务`。因此产品保价、产品降价补差、具体产品优惠承诺、具体产品活动价争议，不再优先归 C端产品或子品牌的 `营销推广`。

```text
【{等级}】【中国区域营销与服务舆情】【智能家居事业群】【中国区域】【内销业务】
```

若内容核心不是价保/营销规则，而是客服拒绝处理、推诿、诱导取消订单、安装/物流服务不当等服务问题，归 `中国区域营销与服务舆情` + `服务售后`。标签冲突时，在AI研判中说明风险性质，不自造标签。若客户后续再次明确要求“具体产品价保归C端/子品牌”，再按新口径另行调整；当前以客户最新版 Word 为准。

## 全屋智能

使用 `全屋智能` 的场景：

- 全屋智能方案、App联动、智能语音、智能家居联动失败。
- 多设备联动、系统平台、智能场景方案争议。

单一产品的智能功能故障，如冰箱App异常、空调联网失败，优先按具体产品线 + `产品质量` 或 `产品售后`。

## 竞对关系

报送场景：

- 美的空调不如格力。
- 美的冰箱被指不耐用、不如竞品。
- 竞品营销、行业排名或市场份额变化直接形成对美的不利影响。
- 美的产品被描述为质量、技术、价格、渠道、服务或份额落后。

标签：

具体产品竞争风险：

```text
【{等级}】【C端产品风险舆情】【智能家居事业群】【{具体业务线}】【竞对关系】
```

泛产品竞争风险：

```text
【{等级}】【C端产品风险舆情】【智能家居事业群】【中国区域】【竞对关系】
```

不报送：

- 普通选购指南、正常参数横评、行业名单罗列。
- “格力美的都不错”“各有优势”“按需选择”。
- 抬高美的的正向对比。

## 子品牌

纳入美的体系的子品牌：

- COLMO
- 小天鹅
- 华凌
- 东芝白电、厨热、微烤等美的体系内东芝产品
- WAHIN
- Coolfree、酷风
- 比佛利、Beverly

默认排除：

- 东芝电视、东芝电视机、TOSHIBA TV、电视系统、电视售后、电视面板、电视渠道。

### 子品牌作为具体产品

只要明确是子品牌，一级标签优先使用 `子品牌风险舆情`，不按普通 C端产品风险处理。例如小天鹅洗衣机故障：

```text
【{等级}】【子品牌风险舆情】【智能家居事业群】【洗衣机】【产品质量】
```

若小天鹅洗衣机故障已经进入维修、保修、换机、退货退款等售后链路，则使用：

```text
【{等级}】【子品牌风险舆情】【智能家居事业群】【洗衣机】【产品售后】
```

### 子品牌品牌层面风险

当负面落在子品牌整体形象、定位、口碑、品牌争议，而不是单一产品功能问题时，使用：

```text
【{等级}】【子品牌风险舆情】【智能家居事业群】【{子品牌业务线}】【品牌形象】
```

建议映射：

- COLMO：`高端品牌`
- 小天鹅：`洗护品牌`
- 华凌：`年轻化性价比品牌`
- 东芝白电/厨热/微烤：按子品牌风险优先；业务线可使用具体产品线或最接近的子品牌业务线，并在摘要说明。

## 美美家园

美美家园是美的相关产业线/服务品牌。

负面或敏感事实涉及美美家园门店、服务、安装、维修、交付、客服、订单履约、售后处理时，优先归：

```text
【{等级}】【中国区域营销与服务舆情】【智能家居事业群】【中国区域】【服务售后】
```

广告、正常介绍、正向导购、无负面事实的品牌露出不报送。
