# 报送判定、等级与高敏

本文件用于判断是否报送、是否重复、危机等级和高敏状态。

## 主体相关性

进入报送判定前，必须先确认内容是否明确指向美的体系。

可进入后续判断的情况：

- 原文明确讨论美的集团、美的品牌、美的体系产品、服务、门店、渠道、官方账号、业务线、关联公司、管理层或产业板块。
- 原文未直接写出集团名，但链接、账号、投诉对象、店铺名、商品名、图片OCR或系统召回字段能证明涉事对象属于美的体系。
- 原文讨论竞品、行业或第三方，但美的是被比较、被评价、被影射或被追责的一方。
- 原文讨论ToB业务或关联产业板块，且能证明属于美的集团客户关注范围。

默认不相关：

- `美的` 只是普通语言片段，如完美的、甜美的、优美的。
- 人名、地名、店名、品牌名相似，但语境不能证明属于美的集团。
- 非美的品牌、店铺或产品投诉仅因系统召回进入。
- 文章只罗列多个企业、历史案例或行业背景，美的不是被评价或批评对象。
- 海外业务正面/中性，且不牵连国内消费者、服务标准、国内外区别对待或集团声誉。

主体不确定时输出 `需人工复核`。

## 报送条件

满足任一条件，且负面/敏感落点明确指向美的，即应报送：

- 消费者描述的具体问题落在美的关联产品、官方渠道、门店、物流、安装、售后、维修、价格权益或活动承诺上。
- 公开投诉、监管受理、消费维权、媒体帮办中，投诉对象或核心事实指向美的关联主体。
- 竞品、评测、导购或行业讨论形成对美的不利结论。
- 讨论美的集团、控股资产、高管、股东利益、资本承诺、经营压力、监管、诉讼、资产处置、亏损或名誉争议。
- 涉及人身安全、公共安全、监管合规、环保、劳动关系、群体投诉或社会敏感议题，且风险落在美的。
- 转发、评论、截图或表情包没有新增事实，但核心表达对美的不利。
- 涉及国内外差异、区别对待、弱势群体受损、公共安全或法律民生节目跟踪。
- 证券社区或财经媒体讨论超出普通股价涨跌，延伸到经营问题、战略投资、业务布局、财报压力、资本承诺、品牌声誉或投资者集中质疑。
- 员工、售后师傅、工厂应聘者或外包人员指向美的体系的欠薪、劳动纠纷或用工争议。
- 高管、创始人、投资并购、年报下降、机构唱空、战略质疑、监管合规、社会责任争议、竞品攻击、黑稿传播等集团重点信息。

## 不报送条件

满足任一条件，且没有其他明确美的风险时，不报送：

- 负面主体是竞品、行业、第三方、城市建设、装修方、平台或个人，美的只是参照或背景。
- 比较关系中美的是更强、更好、领先或正向标尺。
- 美的是历史案例、行业地位、收购背景、供应合作、技术成果、品牌露出、展会露出或榜单参照。
- 内容是导购、参数、教程、招商、广告、好评、产品介绍、获批、合作进展或正向经营成果。
- 普通竞对分析或正常横评，结论是都不错、各有优势、按需选择或差距不大。
- 标题党、痛点铺垫或焦虑引流，正文实际在介绍美的优势。
- 文本过短、OCR噪声、链接缺失或内容混乱导致主体无法确认。
- 仅为股票日常涨跌、盘口波动、K线技术指标、成交额、市值变化、资金流入流出、买卖点提示或短线交易情绪，未延伸到经营、战略、投资、业务或品牌声誉。
- 同一链接、投诉、视频、传播链或状态流转已报送且无新增事实。

## 弱显性风险

即使没有典型负面词，同时满足三项也应报送：

1. 美的是明确涉事主体。
2. 内容描述了异常状态、体验失败、权益受损、质量疑问、服务阻塞、承诺未达成或公众质疑。
3. 普通客户会认为这对美的不利。

## 反误报验证

准备报送前必须回答：

1. 不利含义是否明确落到美的。
2. 美的是否不是地址、背景、历史案例、展会露出、正向参照或普通形容词。
3. 竞品/行业内容中，美的是否确实被负面评价。
4. 集团/资本内容是否存在当前风险，而不是单纯正向进展。
5. AI初始标签提示的风险，能否被标题、摘要和正文独立支持。
6. 美的负面点在原文中的篇幅占比和论述权重是否较低。若只是局部提及，报送摘要和研判是否已体现“少量/局部提及”，避免放大风险。不要机械套用固定比例阈值。

任一答案为否，不能直接报送；改为不报送或人工复核。

## 反漏报验证

准备不报送前必须回答：

1. 是否存在美的为涉事主体的具体异常或权益争议。
2. 是否存在疑问句、短文本、截图、转发等弱表达，但实际在说美的问题。
3. 是否涉及高管、资本、监管、投诉、公共安全、召回、群体影响等敏感事项。
4. 是否因短链、转载、截断或正文缺失导致主体/风险未识别。

任一答案为是，不能直接剔除；改为报送或人工复核。

## 低占比负面提及

当原文主体不是美的，且美的负面点只占较小篇幅或不是全文主线时，不能简单放大为整篇美的负面。

处理规则：

- 美的负面点明确、具体、可独立成立：可以报送，但摘要和AI研判必须标明“原文少量提及/局部提及/作为对比项提到”。
- 美的只是背景、名单、顺带提及，未形成明确不利判断：不报送。
- 美的负面点是否成立依赖截断内容、图片OCR或上下文缺失：需人工复核。

摘要和研判要保留比例感，避免客户误以为通篇内容都在讨论美的负面，从而造成不必要的舆情焦虑。

不要把任何固定百分比作为硬判断标准；应结合正文整体结构、标题和段落重心、负面点出现位置、是否反复论述、是否构成文章主结论来判断摘要表达力度。

## 去重处理

去重键优先级：

1. 投诉编号键，如黑猫、消费保、12315投诉编号。
2. 平台内容ID键，从链接提取唯一ID。
3. 规范化链接键，保留平台关键ID参数。
4. 内容签名键：平台 + 发布人ID + 标题前80字 + 摘要前200字 + 正文前300字。
5. 传播链键：同一原文、视频或话题的转发链核心文本。

处理方式：

- 同一键已报送且无新增事实：`暂不重复报送`。
- 同一投诉状态流转但核心事实不变：`暂不重复报送`，说明为同投诉新进度。
- 同一投诉出现新增事实、金额扩大、诉求升级、媒体介入：可报送为进展更新。
- 同一话题多平台扩散：首发或重点平台可报送，低互动搬运标注同话题传播链。

## 等级判定

按就高原则判定。

### 重大危机

- 单篇微博转评赞超过5000、微信阅读10W+、论坛评论超过3000、小红书转评赞超过1000、抖音转评赞超过10000、视频号转评赞超过1000。
- 成为热搜、微博话题或极可能上升热搜。
- 党报央媒、头部财经、境外媒体集中负面。
- 省级以上政府介入、市监局处罚、约谈调查。
- 2人及以上死亡，或100名以上消费者/员工受影响。
- 严重安全事故、重大法律诉讼、政治敏感、群体健康危害。
- 单起损失超过300万元。
- 跨平台大规模扩散或专项传播链。
- 高敏场景明显传播，如国内外差异、社会公平、公共安全、人身伤害、群体权益。

### 危机

- 微博转评赞500-5000、微信阅读1000-10W、小红书200-1000、抖音2000-10000。
- 多家区域、省级或全国媒体报道。
- 县地级政府警告、检查、整改、消协问询或市监局立案。
- 1人死亡或重伤，或11-100人受影响。
- 核心管理人员或部门涉嫌违法违规。
- 损失100-300万元。
- 权威媒体持续跟进同一事件。
- 集团关联资产持续亏损、甩卖、强负面或投资者集中质疑。

### 异常事件

- 普通用户、自媒体或投诉平台发布负面，暂未大规模传播。
- 微博转评赞低于500、小红书低于200、抖音低于2000、微信低于1000。
- 10名及以下消费者/员工受影响。
- 县级问询、站点转办、一般EHS风险。
- 损失10-100万元。
- 低互动吐槽、短视频、图文笔记。
- 竞品对比明确不利但暂无高传播。

## 高敏判定

高敏独立于等级。触发高敏时在标题最前加 `【高敏❗】`。

必定高敏：

- 爆炸
- 火灾
- 冒烟
- 死亡
- 伤亡
- 触电
- 中毒
- 自燃

原则高敏：

- 315
- 央视
- 人民日报
- 新华社
- 市场监管处罚
- 监管约谈
- 老人儿童弱势群体权益受损
- 法律民生节目跟进

正向安全表述不触发高敏，例如“不用担心中毒”“防爆锅”“安全稳定”“0故障”。
