# 标签格式与标签池

本文件是所有报送标题的统一格式标准。标签池以客户核对后的最新版《副本美的舆情打标规则.docx》为准。生成推送标题前必须先确定场景，再按本文件组装标签。

产品对应业务线以 [product-category-map.md](product-category-map.md) 为优先依据；本文件只定义可用标签和格式。

## 总格式

默认使用五段式：

```text
【{等级}】【{二级标签}】【{三级标签/事业部}】【{四级标签/业务线}】【{五级标签/舆情分类}】
```

高敏时在最前方追加固定前缀：

```text
【高敏❗】【{等级}】【{二级标签}】【{三级标签/事业部}】【{四级标签/业务线}】【{五级标签/舆情分类}】
```

`【高敏❗】` 是固定字符，不得写成 `【高敏】`、`【高敏!】`、`【高敏 ❗】` 或 `【高敏❗️】`。

## 唯一四段式例外

非高管集团层面信息使用四段式：

```text
【{等级}】【集团层面】【品牌整体】【{五级标签/舆情分类}】
```

高敏非高管集团信息：

```text
【高敏❗】【{等级}】【集团层面】【品牌整体】【{五级标签/舆情分类}】
```

只有集团非高管信息可以省略四级业务线。其他场景不得省略标签位。

## 等级

仅可使用：

- `异常事件`
- `危机`
- `重大危机`

涉及高敏情况时，在最前面加 `【高敏❗】`，等级本身仍保留上述三类之一。

## 二级标签池

最新版客户标签池仅使用以下二级标签：

- `中国区域营销与服务舆情`
- `C端产品风险舆情`
- `子品牌风险舆情`
- `B端业务风险舆情`
- `集团层面`

报送时不要因语义临时创造二级标签。旧口径中的 `C端产品口碑舆情`、`C端产品信息舆情`、`B端业务声誉舆情`、`美的集团品牌舆情`、`高敏风险舆情` 等不作为当前标题标签使用。

## 三级标签/事业部池

- `智能家居事业群`
- `工业技术事业部`
- `楼宇科技事业部`
- `机器人与自动化事业部`
- `其他业务`
- `品牌整体`

公司名、品牌名或产品名不得临时提升为事业部。按最新版客户表，万东医疗、锐珂医疗、美的能源、安得智联、美云智数、美智光电均属于 `其他业务` 下的四级业务线。

## 四级标签/业务线池

智能家居事业群：

- `家用空调`
- `洗衣机`
- `冰箱`
- `小家电`
- `厨热`
- `微烤`
- `中国区域`

`中国区域` 是客户标签体系中的四级业务线，主要用于明确属于C端产品或服务链路，但无法分清具体业务线、不包含在表内、或同时涉及多个业务线产品的情况。

B端业务：

- 工业技术事业部：`暖通家电部件`、`新能源汽车部件`、`机器人核心部件`
- 楼宇科技事业部：`电梯`、`暖通`
- 机器人与自动化事业部：`库卡`
- 其他业务：`美的能源`、`锐珂医疗`、`万东医疗`、`安得智联`、`美云智数`、`美智光电`

集团层面：

- 高管相关固定使用 `创始人及高管舆情`
- 非高管集团层面四级位省略，直接进入五级舆情分类

B端业务风险舆情可用：

- `服务售后`

- `营销推广`
- `人资纠纷`

集团层面可用：

- `美的国际`

## 固定格式速查

中国区域营销与服务：

```text
【{等级}】【中国区域营销与服务舆情】【智能家居事业群】【{家用空调/洗衣机/冰箱/小家电/厨热/微烤/中国区域}】【{服务售后/全屋智能/内销业务/美粉及美粉活动相关/政策合规}】
```

C端具体产品：

```text
【{等级}】【C端产品风险舆情】【智能家居事业群】【{具体业务线}】【{产品质量/产品售后/营销推广/人资纠纷/合规诉讼/环保安全/产业生态/竞对关系}】
```

子品牌：

```text
【{等级}】【子品牌风险舆情】【智能家居事业群】【{具体业务线或中国区域}】【{产品质量/产品售后/营销推广/人资纠纷/合规诉讼/环保安全/产业生态/竞对关系}】
```

B端业务：

```text
【{等级}】【B端业务风险舆情】【{事业部}】【{业务线}】【{舆情分类}】
```

集团涉及高管：

```text
【{等级}】【集团层面】【品牌整体】【创始人及高管舆情】【高管名誉】
```

集团非高管：

```text
【{等级}】【集团层面】【品牌整体】【{舆情分类}】
```

## 关键规则

命中多个C端风险时，按以下优先级取最高：

```text
服务售后 > 产品售后 > 产品质量
```

服务售后：

- 乱收费、服务态度、推诿、安装、物流、客服处理等服务问题。
- 二级标签使用 `中国区域营销与服务舆情`。

产品售后：

- 产品故障后有维修、换新、保修、退货退款、售后上门、多次维修未果等诉求或处理链路。
- 二级标签使用 `C端产品风险舆情` 或 `子品牌风险舆情`。

产品质量：

- 纯产品质量问题，尚未进入售后处理链路。
- 二级标签使用 `C端产品风险舆情` 或 `子品牌风险舆情`。

价保/营销类：

- 价保、降价补差、补贴、活动承诺、优惠券、活动价争议等，按最新版客户表归 `中国区域营销与服务舆情`，五级标签统一使用 `内销业务`。
- 若客户后续单独要求按具体产品归 C端/子品牌，再另行校准；在当前最新版 Word 口径下，以 `中国区域营销与服务舆情` + `内销业务` 为准。

## 产品线边界

- 烤箱、微波炉、蒸烤箱、微蒸烤一体机、水波炉归 `微烤`。
- 热水器、燃气热水器、烟机、灶具、洗碗机、集成灶、净水机、管线机归 `厨热`。
- 下表没有的C端产品、无法分清业务线、或同时涉及多个业务线产品时，四级业务线使用 `中国区域`。

## 禁止项

- 不得输出旧写法 `售后服务`；最终标签统一改为 `服务售后`。
- 服务售后不得写成 `【C端产品风险舆情】...【服务售后】`；二级标签统一使用 `中国区域营销与服务舆情`。
- 明确子品牌不得写成普通 `C端产品风险舆情`；二级标签优先使用 `子品牌风险舆情`。
- 产品质量导致的售后不得继续写 `产品质量`；进入售后处理链路时使用 `产品售后`。
- 价保/营销类不得优先写成 `C端产品风险舆情` 或 `子品牌风险舆情`；当前最新版客户表要求归 `中国区域营销与服务舆情` + `内销业务`。
- 不得把 `创始人及高管舆情` 当成高管标签的唯一末级；高管固定同时包含 `创始人及高管舆情` 和 `高管名誉`。
- 不得把万东医疗、锐珂医疗、安得智联、美云智数、美的能源、美智光电等公司名临时提升为事业部。
- 不得把 `KUKA`、`Swisslog`、`瑞仕格` 输出为四级业务线；当前客户表四级业务线统一使用 `库卡`。
- 标签池没有完全匹配时，选择最接近的客户固定标签，把细节写入摘要、AI总结或AI研判。
